Wandu is PHP7 Framework.

## Installation

```sh
$ cd /your/project/path
$ composer require wandu/framework
$ vendor/bin/wandu install
```

then, there are 2 questions.

```
 install path? [/your/project/path]:
 >

 app namespace? [Wandu\App]:
 >
```
